Selecting

Select an element by class name

$( ".class-name" )

Select an element by ID

$( "#idName" )

Select an element by type

$( "article" )

Select a child element

$( "article .test" )

Select an element by data attribute

$( "[data-attribute='value']" )

More about jQuery selectors

Manipulating

Apply CSS

$( ".test" ).css( "color", "#fff" );
$( ".test" ).css( "backgroundColor", "#000" );

Change the HTML inside the selected element

$( "article" ).html( "<h2>New title</h2>" );

Remove an element

$( "article h2" ).remove()

Change the value

$( "input" ).val( "new value" );
